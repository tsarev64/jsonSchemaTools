Интерфейс разделён на две вкладки:

Генерация / Проверка — работа с одной схемой и сообщением.
Сравнение схем — анализ различий между двумя схемами.

Все текстовые поля поддерживают нумерацию строк и прокрутку. Ошибки и результаты отображаются в нижней части окна с возможностью клика для перехода к проблемному месту.

ВКЛАДКА «ГЕНЕРАЦИЯ / ПРОВЕРКА»

Загрузка JSON-схемы
Нажмите кнопку «Открыть...» рядом с полем «Файл схемы».
Выберите файл .json, содержащий корректную JSON Schema.
Содержимое файла автоматически загрузится в левое текстовое поле.
Генерация примера
Убедитесь, что схема загружена.
Нажмите «Сгенерировать пример по схеме».
Программа создаст пример JSON-объекта, соответствующий схеме, и отобразит его в правом поле («Сообщение / сгенерированный пример»).
Проверка сообщения
Вставьте JSON-сообщение в правое поле.
Нажмите «Проверить сообщение».
Если сообщение соответствует схеме — появится уведомление об успехе.
При ошибках:
Внизу отобразится список ошибок с указанием пути (например, $.user.email).
Появится подробное модальное окно с описанием ошибки и рекомендациями.
Клик по любой строке в списке ошибок подсветит соответствующее поле в схеме и сообщении.
Сохранение сообщения
После генерации или редактирования сообщения нажмите «Сохранить сообщение».
Выберите путь и имя файла (по умолчанию — .json).

ВКЛАДКА «СРАВНЕНИЕ СХЕМ»

Загрузка двух схем
Используйте кнопки «Открыть...» для загрузки Схемы 1 и Схемы 2.
Содержимое отобразится в левых и правых панелях соответственно.
Сравнение
Нажмите «Сравнить схемы».
Программа выполнит семантическое сравнение (не строчное!), учитывая структуру JSON Schema:
Добавленные/удалённые свойства
Изменения типов, required, enum, anyOf и т.д.
Изменения вложенных объектов и массивов
Результаты
Отличия отображаются в нижнем окне с цветовой маркировкой:
Зелёный — добавлено
Красный — удалено
Коричневый — изменено
Клик по строке с отличием подсветит соответствующее поле в одной или обеих схемах.
Копирование результатов
Нажмите «Копировать результат», чтобы скопировать весь отчёт в буфер обмена.
